# Movie Store

> Movie store is the Application that shows recent movies from movie API store 

## Built With

- Reactjs
- Tailwind CSS
- RapidAPI



## Getting Started

**clone the project by running:

```
$ git clone https://github.com/JoffreyNK/movie-website

cd movie-website

npm install

npm start
```
